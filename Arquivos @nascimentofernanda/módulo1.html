2. O que são clients

Clients (palavra em inglês para clientes) são todos os usuários da aplicação Web que se conectam às aplicações através da internet, mais precisamente dos navegadores.

A experiência dos clientes com a aplicação depende de diversos fatores como, por exemplo: velocidade da conexão, configuração da responsividade, etc.

A responsividade ou design responsivo consiste em criar o layout do website de acordo com o tipo de aparelho no qual ele será exibido.

A compatibilidade dos atributos das TAGs com a versão de cada navegador também influencia na experiência do usuário.

O que são navegadores?

Também chamados de browsers navegadores não programas gratuitos criados por empresas e utilizados para abrir e executar arquivos.

O que são aplicações Web?

Aplicações Web são soluções que possuem a internet como meio de comunicação entre o client(cliente) e o server(servidor) sem necessidade de instalação.

Aplicações Web são diferentes de aplicativos.

3. O que são servers?

Servers (palavra em inglês para servidores) são os locais que armazenam os dados que são acessados pelos clientes através das aplicações Web.

Existem os servidores locais, que são acessados apenas pela rede de intranet local e os servidores globais.

Os servidores podem ser de arquivos, de segurança, de streaming, de e-mail e Web.

Componentes dos servidores:

Os servidores são compostos de softwares e hardwares.

Softwares:

  • Sistemas Operacionais
  • Monitoramento
  • Servidores Web

Hardwares:

  • Armazenamento (HDs e SSDs)
  • Memória
  • Processadores

Os servidores recebem requisições HTTP e devolvem aplicações em HTML, CSS, Javascript, Texto, etc.

Para que o seu site seja publicado e visível na internet é necessário que ele seja hospedado por um servidor de sites.

4. Sobre linguagens de programação web:

O que é linguagem de programação?

Linguagens de programação são linguagens que especificam um conjunto de instruções e regras usadas para gerar programas.

As linguagens possuem características diferentes para aplicações executadas para clientes e aplicações para servidores.

Cada linguagem tem sua característica mesmo que possuam a mesma finalidade.

HTML é linguagem de programação?

Não! O HTML é uma linguagem de marcação que utiliza textos delimitados por nomes que são interpretados pelo navegador. Cada delimitador possui uma função e um comportamento específico.
