Estrutura HTML

Um arquivo HTML pode ser criado à partir de qualquer editor de texto, mas para facilitar, são utilizadas IDEs.

Qualquer página pode ter seu código HTML exibido à partir do Inspetor de elementos, basta abrir a página desejada e apertar a tecla F12 que se abrirá uma janela com o código-fonte HTML de cada elemento.

Tags

O HTML é constituído basicamente de Tags de abertura e fechamento que "formatam" os componentes da página.

Toda página HTML deve conter as tags: HTML, HEAD e BODY.

A tag HTML é utilizada para informar ao navegador que aquele é um arquivo HTML.

A tag HEAD é utilizada no início do documento e nela constam todas as informações importantes sobre o mesmo.

A tag BODY é a tag na qual estão contidos todos os elementos visuais do arquivo, ou seja, tudo aquilo que será exibido ao usuário no navegador.

Algumas tags muito utilizadas no HTML são:

< a > tag utilizada para adicionar links à página. Possui como atributo obrigatório o href="" que especifica o link para onde o usuário será redirecionado.

< b > tag utilizada para deixar o conteúdo dentro dela em negrito.

< i > tag utilizada para deixar o conteúdo dentro dela em itálico.

< u > tag utilizada para deixar o conteúdo dentro dela sublinhado.

< h1 > até < h6 > tags utilizadas para dar destaque aos títulos e subtítulos. O tamanho do texto é diminuído gradualmente conforme o número aumenta.

< p > tag utilizada para escrever um parágrafo.

< mark > tag utilizada para deixar o texto marcado como um marca-textos.

< small > tag utilizada para escrever um texto em fonte pequena.

< ol > tag utilizada para criar uma lista ordenada. É utilizada junto com a tag < list > .

< ul > tag utilizada para criar uma lista não ordenada. Também utilizada com a tag < list >.
