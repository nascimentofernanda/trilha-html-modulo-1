Objetivo Geral

Aprender a como criar um formulário.

O que são formulários?

Formulários são recursos com campos preenchíveis utilizados para coleta de dados.

No HTML os formulários são criados utilizando a tag pai < form > em conjunto com suas tag filhos escolhidas de acordo com a necessidade do site.


Exemplo de formulário sem formatação CSS

*Na IDE digite sem os espaços antes dos sinais de abertura e fechamento < e >.

        < form > 
        < label >Nome: < /label >< input type="text" name="nome" />< br >
        < label >Idade: < /label >< input type="number" name="age" />< br >
        < label >Senha < /label >< input type="password" name="password" />
        < /form >

Exemplo de formulário formatado

Algumas tags e labels importantes para o formulário:

Tag form: "Marca" o início e o final do formulário, todos os campos e atributos do formulário devem estar dentro dessa tag

Tag input: Acrescenta os campos onde o usuário irá digitar os dados. Alguns labels importantes da tag input são:

  • type: Define o tipo de dado que será digitado no campo (texto, numérico, senha, etc...)
  • name: Define o nome do campo, auxiliando a identificação para formatação e utilização de Javascript no futuro.

Tag button: Cria um botão para a submissão do formulário.
