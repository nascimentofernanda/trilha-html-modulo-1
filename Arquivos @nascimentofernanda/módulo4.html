Objetivo Geral

Aprender a estruturar a página HTML criando divisões de conteúdo para facilitar a formatação, bem como a formatar com algumas tags próprias.

Alguns exemplos de tags de formatação:

  • Tag < b >: Utilizada para deixar o texto em negrito.
  • Tag < i >: Utilizada para deixar o texto em itálico.
  • Tag < u >: Utilizada para sublinhar um texto.
  • Tag < mark >: Utilizada para marcar o texto como um marca texto.
  • Tag < sup >: Utilizada para sobrescrito.
  • Tag < sub >: Utilizada para subescrito.
  • Tag < bloquote >: Utlizada para fazer uma citação.
  • Tag < font >: Utilizada para formatar a fonte utilizada no texto. Necessita dos atributos definidos para que funcione.

Tags de acessibilidade:

  • Tag strong: Utilizada quando além de deixar o texto dentro da tag em negrito, o desenvolvedor deseja que o texto tenha uma "conotação" diferente quando a página for acessada por pessoas que possuem deficiência visual.

Tags de estruturação:

  • Tag div: Usada para estruturar "tópicos" separando o texto em "caixas" que serão identificadas por id ou class.
  • Tag span: Mesma funcionalidade da tag div, porém, a tag div é display block, enquanto que a tag span é display inline.
  • Tag fieldset: "Segmenta partes do formulário, utilizada com a tag legend.
