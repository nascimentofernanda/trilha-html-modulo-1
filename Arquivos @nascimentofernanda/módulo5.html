Trilha HTML Developer DIO

Objetivo Geral

Aprender a adicionar arquivos multimídia na página web.

Algumas Tags:

  • Tag img: Utilizada para adicionar imagens tanto de arquivos locais como de páginas web. Possui diversos atributos, o mais importante é o "src" que indica o local da imagem, além do "alt" que auxilia a leitura de pessoas deficientes visuais.
  • Tag audio: Utilizada para adicionar arquivos de som do computador para a página web com o auxílio da tag filha source.
  • Tag video: Utilizada para adicionar videos. Funciona da mesma forma que a tag audio.
  • Tag iframe: Utilizada para adicionar arquivos de mídia de outras páginas na sua página. Ex: Um vídeo do youtube, um mapa do maps, etc...

Para acessar todas as tags e atributos das mesmas confira o site W3 HTML Element Reference
