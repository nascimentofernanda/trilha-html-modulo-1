O que são tabelas e para que servem?

Tabelas são um conjunto de linhas e colunas que são utilizadas geralmente para exibição de conjunto de dados relacionados.

O encontro de uma linha com uma coluna é chamado de célula.

Para criar uma tabela é necessário utilizar a tag table para dar início à tabela editando seus atributos, em conjunto com as tags de linha e coluna.

Algumas tags utilizadas em tabelas:

  • Tag tr:Significa table row e cria linhas na tabela.
  • Tad td:
    • Cria uma célula.
  • Tad th: Identifica o cabeçalho da tabela. Essencial para leitura de acessibilidade.
  • Tag tbody: Agrupa todas as células que fazem parte do corpo da tabela.
  • Tag thead: Assim como a tag th identifica o cabeçalho da tabela.
  • Tag tfoot: Identifica o rodapé da tabela.
  • Tag caption: Nome da tabela.

Para verificar todos as tags e atributos disponíveis para tabelas no HTML visite a página: HTML < table > tag.
